Project Objectives:

Project 1(Retail Customer Transactions Data): Objective of this project is to analyze data for a retail store and find customer buying habits, the sales potential of different categories & tracking purchases/returns of different product categories.

Project 2 (Mexican Restaurant Data): Analyze interesting patterns and customer preferences to help understand market and make investment decisions, visualize data using Strategic Dashboards.

Project 3 (Covid-19 Data) : Provided by Ourdatain world, objective of this project was to get insights about infection trends and casualties caused by covid -19 around the world. Which countries or locations are more successful in controlling covid-19.

About Data:

Project 1: Retail Customer Transactions Data: This Data provided by Kaggle contain three tables (Customer, Transactions, product_information). Transaction Data contains columns like transaction Id, cust_Id, product_code Qty, Date, amount etc .It is can be connected to the customer table using cust_Id as a key. Further product_information table containg product category and sub category code and columns for respective products(Men,women,mobile...).

Project 2: Maven Analystics, Mexican Resturant Data. This data includes 5 tables : Consumers,Consumer Preference,Resturant,Resturant cusine & Ratings.

Project 3: Covid-19 Data : This data is provided by (Our World in Data) till Spetember 2021. It contains profiles of 207 countries, Which contain information like total death, new cases, testing, location.

Python Libraries : Pandas, Numpy, Seaborn, Matplotlip

Retail Customer Transactions Data

  • After combining transaction and product info table and adding features like weekend/weekday columns. It can be determined which days, month, a year has more transactions or sale. As well as seasonal variation in the sale.
  • After grouping data by product category & sub-category, product-based transactions variation can be determined. In this data Home Kitchen, Electronics are the highest source of sale followed by clothing and footwear. These insights will be critical for maintaining inventory.
  • The third important insight that could be found is through Sale/Return feature added to data. It is useful in determining which product has a higher tendency of being returned like in this data clothing Women, Tools, Personal appliances have a count of 10-11 returns per 100 sales transactions. Important to understand this in order to reduce losses.

Mexican Resturant Data Insights for Investment Decision

  • Consumer data, Rating data, and Restaurant details data can be combined using consumer ID & Restaurant ID as a key. Data obtained can be used to detrmine various insights Customer age, factors affecting resturant rating.
  • For these 130 resturants having facilities like bar, smoking area,parking increase Rating.Also, frenchised resturants have heigher rating than non-frenchised.
  • City of Luis Potos has maximum customer share(66.15%), followed by Tamaulipas 19.23% customer share and 23 resturants and Morelos customer share 20.80% and 23 resturants.
  • Opening medium budget resturant with Mexcian cusine having facilities like smoking area and bar will attract major share of customer.

Covid-19 Data Anlysis of Infection Trend

  • Checked casualties count by covid -19 virus of 207 countries on the world map, along with total cases.
  • After differentiating locations like countries and continent seprarately, Infection by continent can be checked till Sepetember 2021 Asia and Europe are on top of the list with 58.74 Million & 52.11 Million total infections. Europe still has more death count 1.13 Millions
  • Infection rate can be calculated by dividing the total population of a country by Total infections. As this data is a Time series data calculated infection rate can be tracked Over the months and show visually on Tableau.
